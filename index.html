Panduan Lengkap HTML5 Semantik

Dipublikasikan oleh: Admin pada 14 Juli 2026

Pendahuluan

Saat ini, menulis kode web bukan sekadar membuat tampilan yang bagus. Kita harus menggunakan HTML semantik agar website kita lebih mudah dikenali oleh mesin pencari seperti Google.

Seorang ahli web pernah berkata, Gunakan tag yang tepat untuk pekerjaan yang tepat.

Mengapa Menggunakan Tag Semantik?

Sebelum HTML5, kita sering terjebak dalam "div soup" (terlalu banyak menggunakan tag div). Sekarang, aturan tersebut sudah diperbarui.

Cara lama membuat struktur layout adalah dengan dan menggunakan tag <div id="header">, namun sekarang kita wajib menggunakan tag <header> secara langsung.

Mari kita definisikan satu istilah: Semantic Markup adalah metode penulisan kode HTML di mana tag yang digunakan merepresentasikan arti dari konten tersebut.

"Elemen semantik secara jelas mendeskripsikan maknanya baik kepada browser maupun kepada developer."

Kutipan di atas diambil langsung dari dokumentasi resmi W3C HTML5 Specification.
