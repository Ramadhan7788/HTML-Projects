# Basic HTML Website

A simple multi-page website built using **HTML5 only** as part of the roadmap.sh Basic HTML Website project.

This project focuses on creating a well-structured website using semantic HTML without any CSS or JavaScript. The main objective is to practice organizing web pages, improving accessibility, and preparing a solid foundation for future styling.

## Project Pages

- Home
- Projects
- Articles
- Contact

Each page includes a consistent navigation bar for easy navigation throughout the website.

## Features

- Semantic HTML5 structure
- Multi-page website
- Navigation bar on every page
- SEO-friendly meta tags
- Open Graph meta tags for social media sharing
- Accessible HTML elements (labels, alt text, headings, etc.)
- Clean and maintainable HTML structure

## Technologies

- HTML5


## Learning Goals

This project was created to practice:

- Creating a multi-page website
- Using semantic HTML elements
- Building a consistent page structure
- Writing SEO-friendly HTML
- Preparing HTML for future CSS styling
