# Blog Post Page

A simple HTML project created to complete the **Blog Post Page** challenge from **roadmap.sh**.

## About

This project focuses on building a semantic and accessible blog post page using only HTML.

## Features

* Semantic HTML structure using `<header>`, `<main>`, `<article>`, and `<footer>`
* Proper heading hierarchy with a single `<h1>` and nested `<h2>` sections
* Semantic content elements, including:
  * `<p>`
  * `<ul>`
  * `<blockquote>`
  * `<code>`
  * `<pre>`
* A `<figure>` containing an image with a descriptive `alt` attribute and `<figcaption>`
* SEO-friendly metadata, including:
  * `<title>`
  * `<meta charset>`
  * `<meta name="viewport">`
  * `<meta name="description">`
* Open Graph metadata for improved social media sharing
