# Contact Form

A simple HTML project created to complete the **Contact Form** challenge from **roadmap.sh**.

## About

This project focuses on building an accessible and semantic contact form using only HTML.

## Features

* Semantic HTML structure
* Proper `<form>` element with `action` and `method="post"`
* Associated `<label>` elements for every form control
* Required fields with built-in browser validation
* Email validation using `type="email"`
* Subject selection using a `<select>` element
* Message field using a `<textarea>` with a minimum length
* Radio button group wrapped in a `<fieldset>` with a `<legend>`
* Optional newsletter subscription using a checkbox
* Submit button with clear text
* Responsive viewport and basic metadata

## Technologies Used

* HTML5
