# Photo Showcase

A simple HTML project created to complete the **Photo Showcase** challenge from **roadmap.sh**.

## About

This project focuses on building a semantic HTML page that showcases images and video while following accessibility best practices.

The page demonstrates how to use images with descriptive alternative text, captions, and embedded media using only HTML.

## Features

* Semantic HTML structure (`<header>`, `<main>`, `<footer>`)
* Six images with descriptive `alt` attributes
* `width` and `height` attributes on every image to reserve layout space
* Images wrapped in `<figure>` with `<figcaption>`
* One decorative image using `alt=""`
* Embedded video with playback controls
* Video poster image and fallback text
* Proper document metadata (`<title>`, `charset`, and `viewport`)

## Learning Objectives

Through this project, I practiced:

* Writing semantic HTML
* Using meaningful `alt` text for accessibility
* Understanding the difference between `alt` text and `figcaption`
* Reserving image space with `width` and `height`
* Embedding videos with the `<video>` element
* Organizing media files in a simple project structure
