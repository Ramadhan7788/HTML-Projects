Tag HTML Dasar

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. Berikut adalah beberapa tag HTML dasar:

  • <h1> hingga <h6>: Tag heading untuk judul.
  • <p>: Tag paragraf untuk teks.
  • <a>: Tag anchor untuk membuat tautan.
  • <img>: Tag gambar untuk menampilkan gambar.
  • <ul> dan <ol>: Tag daftar tidak berurutan dan berurutan.

Contoh Penggunaan Tag

Berikut adalah contoh penggunaan beberapa tag HTML:

                
                <h1>Judul Utama</h1>

                <p>Ini adalah paragraf pertama.</p>

Informasi Tambahan

HTML memiliki banyak tag lainnya yang dapat digunakan untuk berbagai keperluan, seperti form, tabel, dan multimedia.
