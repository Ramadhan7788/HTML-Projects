# Pricing Comparison Table

A simple HTML project created to complete the **Pricing Comparison Table** challenge from **roadmap.sh**.

## About

This project focuses on building an accessible and semantic pricing comparison table using only HTML.

## Features

* Semantic HTML structure
* A `<table>` with `<thead>` and `<tbody>` to separate header rows from data rows
* A `<caption>` as the first child of the table to describe the comparison
* Proper header scopes using `scope="col"` and `scope="row"` for better accessibility
* At least one merged cell using `colspan` or `rowspan`
* Essential HTML metadata, including:

  * `<title>`
  * `<meta charset>`
  * `<meta name="viewport">`
