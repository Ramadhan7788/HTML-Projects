# HTML-Projects
# Single-Page CV

A simple single-page CV built using **HTML only** as part of the [roadmap.sh Single-Page CV](https://roadmap.sh/projects/single-page-cv) project.

## About

This project was created to complete the **Single-Page CV** challenge from roadmap.sh while practicing the fundamentals of HTML.

The page includes:

- Semantic HTML structure
- SEO meta tags
- Open Graph (OG) meta tags
- Favicon
- A clean and simple single-page CV layout

## Technologies Used

- HTML5

## Project Purpose

This project is intended for:

- Learning semantic HTML
- Understanding basic SEO meta tags
- Implementing Open Graph tags
- Building a simple portfolio project

## Disclaimer

The personal information, education, work experience, contact details, and social media links included in this project are **fictional or randomly generated** for demonstration purposes only.

This project is created **solely for learning and portfolio purposes**.
